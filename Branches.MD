# Git Branch

<img src="./images/branches.svg" alt="branches" />

## What is a branch ?

Git branches is a pointer to a special state of your changes that represent them like :

-  new feature
-  new Version
-  a bug fix
-  ...

## How to use

-  See list of local branches

   ```bash
   git branch
   ```

-  See list of remote branches

   ```bash
   git branch -a
   ```
